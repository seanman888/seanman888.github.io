自訂網域並使用 Hexo 架站

2025-01-03

前言

Hexo 是一個快速上手的靜態網站生成器，特別適合用於建立部落格或個人網站。透過 Hexo clean、Hexo generate、Hexo deploy 幾個指令就能將靜態網站部署到 Github Pages上。

我最喜歡的點是只要有 markdown 撰寫的文章，就可以到 Hexo 社群找到不同風格的框架套用，目前使用的是 Claudia，簡約黑白風。

不過既然要有自己專屬的部落格就要有自己專屬的網域，以下簡單分享從 bluehost 購買域名後替換到 gituhub Page 的過程 :

將域名綁定到gitpage上

  1. 由於之前已經有在 bluehost 上購買過自己想要的域名了，直接前往bluehost Domain 處做設定。

  2. DNS record 用於儲存每個網域名稱對應的相關資訊，這邊透過 CNAME Type將子域名 www.seanmeow.com 指向 seanman888.github.io。TTL設定15分鐘，代表伺服器在 15 分鐘後才會更新該記錄。

  3. 回到github Settings -> Pages 上將子域名輸入至 Custom domain 中，輸入www.seanmeow.com 即會導向 github page 的畫面
